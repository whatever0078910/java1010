JAVA Functions

There are a number of various JAVA functions and methods. The main methods consists of public, which allows public access, static, which means no instance needs to be formed before it is used, void which means it doesnt return any value like 0 or 1 when it completes, main which gets started when a JAVA Virtual Machine (JVM) starts a standalone application, a string, which means configuration parameters being passed in as arguments, and args, which are the cofniguration parameters which are passed into the program.
